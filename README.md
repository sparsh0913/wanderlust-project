#  Wanderlust — Airbnb Clone

Wanderlust is a full-stack Airbnb-inspired web application where users can explore travel listings, create their own listings, and book unique stays around the world. The project focuses on building a scalable booking platform with authentication, CRUD operations, and responsive UI.

---

## Features

*  User Authentication (Login / Register)
*  Create, Edit, and Delete Listings
*  Image Upload for Listings
*  Browse and Search Listings
*  Location-based Listings
*  Reviews and Ratings
*  Responsive Design
*  RESTful API Architecture

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/wanderlust.git
```

2. Navigate to project folder

```bash
cd wanderlust
```

3. Install dependencies

```bash
npm install
```

4. Create `.env` file and add:

```
MONGO_URI=your_mongodb_connection
CLOUDINARY_KEY=your_key
CLOUDINARY_SECRET=your_secret
SESSION_SECRET=your_secret
```

5. Run the server

```bash
npm start
```

6. Open in browser

```
http://localhost:3000
```
